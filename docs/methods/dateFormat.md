# dateFormat

**废弃，推荐使用[dayjs](https://day.js.org/en/)**

## Since

`0.0.1`

## Arguments

- `date: string | number`: The array to convert
- `format: string`: yyyy, M, MM, dd, d, HH, H, mm, m, ss, s

## Returns

- `Returns: string`: The array to convert。

## Exampless

```js
dateFormat(1548297785895, 'yyyy-MM-dd HH:mm:ss');

// 2019-01-24 10:43:05
```
